# Quickstart: Initial Page Setup

## Prerequisites
- Node.js (v18+ recommended)
- pnpm or npm

## Install dependencies
```sh
pnpm install
# or
npm install
```

## Run the development server
```sh
pnpm dev
# or
npm run dev
```

## Project Structure
- `src/` — React app source code
- `specs/1-initial-page-setup/` — Feature specs, plan, research, data model, contracts

## Testing
```sh
pnpm test
# or
npm test
```

## Linting & Formatting
```sh
pnpm lint
# or
npm run lint
```

## Notes
- The app seeds fixture data to localStorage on first load.
- All UI is accessible and keyboard-navigable.
- See `specs/1-initial-page-setup/` for design, data model, and API contracts.
